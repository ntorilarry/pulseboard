# Pulseboard

A modern dashboard application built with Next.js, featuring task management, analytics, and a responsive design with dark mode support.

## Setup

1. Install dependencies:
```bash
npm i
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

That's it! The application will be running locally.
